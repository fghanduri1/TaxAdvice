About This website acts as a financial advisor for users. Users can click on Income and it would direct them to a page that the user can find their income tax.
Income:
This page would calculate the income tax for the users. The user would first pick out if they are single or married. With this the user would get an extra allowance of 907 GBP. The user is also able to change the currency from GBP to USD using a small switch. This is a direct exchange with a value of 0.8 GBP to 1 USD. The user can also enter their yeary income using two sliders where one of the sliders is in single 1k value per point, and the second increases in the 10k value per point. The amount that the user earns is then calculates as the taxable income and this is then calculated for the user and would appear at the bottom in a whold yearly amount and in a monthly amount. The user can also see the income range they are in in this section.
